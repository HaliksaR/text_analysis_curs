# README
Программа calcFrequency производит анализ слов в исходном тексте. 
## Make 
```make
    make
    make open
    or add -no-pie
```
## Требуемые пакеты
### Ubuntu

```make
    apt install libgtk-3-dev
```
### Solus 

```make
    eopkg install libgtk-3-dev
```
